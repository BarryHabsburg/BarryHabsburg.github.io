Question 6.2
When running an MCMC algorithm, it is important to know when the transient (or burn-in) period has finished; otherwise, steady-state statistical analyses such as those in Section 4.3.2 may not be applicable. In practice this often done via a visual inspection of the sample path. As an example, run the random wlak sampler with normal terget distribution $\N(\mu=10,\sigma^{2}=1)$ and proposal $Y~\N(\mu=x,\sigma^{2}=0.01)$. Take a sample size of $N=5000$. Determine roughtly when the process reaches stationarity.

N=5000; sample=np.zeros((N,1));
sigma=0.1; X=sigma*np.random.randn();

for i in range(N):
    Y=X+np.random.randn()*sigma;
    if np.random.uniform(low=0.0,high=1.0)<min(np.exp(-0.5*(Y-10)**2+0.5*(X-10)**2),1):

    sample[i]=X;

plt.plot(sample);
plt.xlabel('Iteration t');
plt.ylabel('State of the chain X_t');
plt.grid()
plt.show()

Roughly at $t=1000$, the random walk sampler became stationary.

Question 6.12
Let $X_{1},\ldots,X_{n}$ be a random sample form the $\N(\mu,\sigma^{2})$ distribution. Consider the following Bayesian model:

  • $f(\mu,\sigma^(2))=1/\sigma^{2}$

  • $(x_{t}|\mu,\sigma)~\N(\mu,\sigma^{2})$, $i=1,\ldots,n$ independently.

Note that the prior for $(\mu,\sigma^{2})$ is improper. That is, it is not a pdf in itself, but by obstinately applying Baayes' formula, it does yield a proper posterior pdf. In some sense it conveys the least amount information about $\mu$ and $\sigma^{2}$. Let $x=(x_{1},\ldots,x_{n})$ represent the data. The posterior pdf is given by

$$f(\mu,\sigma^{2}|x)=\big(2\pi \sigma^{2}\big)\text{exp}\bigg\{ -\frac{1}{2} \frac{\sum_{i}(x_{i}-\mu)^{2}}{\sigma^{2}}\bigg\}\frac{1}{\sigma^{2}}$$

We wish to sample from this distribution via the Gibbs sampler.
(a) Show that $(\mu|\sigma^{2},x) \thicksim \N(\bar{x},\sigma^{2}/n)$, where $\bar{x}$ is the sample mean.
(b) Prove that $$f(\sigma^{2}|\mu,x) \propto \frac{1}{(\sigma^{2})^{n/2+1}} \text{exp}\big( -\frac{n}{2} \frac{V_{/mu}}{\sigma^{2}} \big)$$

(c) Important a Gibbs sampler to sample from the posterior distribution, taking $n=100$. Run the sampler for $10^{5}$ iterations. Plot the histograms of $f(\mu|x)$ and $f(\sigma^{2}|x)$ and find the sample means of these posteriors. Compare them with the classicia estimates.
(d) Show that the true posterior pdf of $\mu$ given the data is given by $$f(\mu|x) \propto \big((\mu - \bar{x})^{2}+V\big)^{-n/2},$$ where $V=\sum_{i} (x_{i}-\bar{x})^{2}/n$. (Hint: in order to evaluate the integral $$f(\mu|x)=\int_{0}^{\infty} f(\mu,\sigma^{2}|x) d\sigma^{2}$$ where it first as $(2\pi)^{-n/2} \int_{0}^{\infty} t^{(n/2)-1} \text{exp}\big(-\frac{1}{2}tc\big)dt$, where $V=nV_{\mu}$, by applying the change of variables $t=1/\sigma^{2}$. Show that the latter integral is proportional to $e^{-n/2}$. Finally, apply the decomposition $V_{\mu}=(\bar{x}-mu)^{2}+V$.)
     Solution (a):

$$f(\mu|\sigma^{2},x) \propto \text{exp}\bigg\{ -\frac{n\mu^{2}\sum_{i} x_{i} + \sum_{i} x_{i}^{2}}{2\sigma^{2}}\bigg\} \propto \text{exp}\bigg\{ -\frac{\mu^{2}-2\mu \sum_{i} x_{i}/n}{2\sigma^{2}/n} \bigg\} \propto \text{exp}\bigg\{ -\frac{(\mu-\bar{x})^{2}}{2\sigma^{2}/n} \bigg\}$$

Which means $(\mu|\bar{x},\sigma^{2})\thicksim\N(\bar{x},\frac{\sigma^{2}}{n})$. $\square$

     Solution (b):

Recollect the posterior probability density function (pdf),

$$f(\sigma^{2}|\mu,x)=(2\pi)^{-n/2}(\sigma^{2})^{-(n/2)-1}\text{exp}\bigg\{ -\frac{V_{\mu}}{2\sigma^{2}/n} \bigg\},$$

$(1/\sigma^{2}|V_{\mu})$ is a Gamma distribution, $\text{Gamma}(n/2,nV_{\mu}/2)$. $\square$

     Solution (c):

The following Python script is an Gibbs sampler, which draws a random sample of size $100$ from a normal distribution and from its posterior distribution with said Gibbs sampler with hundred-thousand samples.

N=10**5; lam=0.2; x=1; sample=np.zeros((N,1));

for i in range(N):
    y=-np.log(np.random.uniform())/lam;
    if np.random.uniform()<min(np.exp((lam-1)*y+(1-lam)*x),1):
        x=y;
    sample[i]=x;

[counts,bins]=np.histogram(sample,bins=100);
x0=np.linspace(0,10,1000);

g=plt.hist(bins[:-1],bins,weights=counts)

     Solution (d):

$$f(\mu|x)=\int_{0}^{\infty} f(\mu,\sigma^{2}|x)d\sigma^{2}= (2\pi)^{-n/2} \int_{0}^{\infty} (\sigma^{2})^{-(n/2)-1}\text{exp}\bigg\{ -\frac{V_{\mu}}{2\sigma^{2}/n} \bigg\} d\sigma^{2}$$

$$(2\pi)^{-n/2} \int_{0}^{\infty} t^{-(n/2)-1}\text{exp}\bigg\{ -\frac{n}{2} V_{\mu}t\bigg\} dt = (2\pi)^{-n/2}c^{-n/2}\Gamma(n/2) \int_{0}^{\infty} \frac{c^{n/2}t^{(n/2)-1}\text{exp}(-\frac{c}{2}t)}{\Gamma(n/2)}dt$$

$$=(2\pi)^{-n/2}c^{-n/2}\Gamma(n/2)$$

Therefore by substituting back $c=nV_{\mu}$ with $V_{\mu}=(\mu-\bar{x})^{2}+V$, we attain,

$$f(\mu|x) \propto (V_{\mu})^{-(n/2)}=\big((\bar{x}-\mu)^{2}+V\big)^{-(n/2)} \propto \bigg( \frac{(\mu-\bar{x})^{2}}{V}+1 \bigg)^{-(n/2)}. \square$$
